# Kanban-Board
A responsive Kanban Board built with HTML, CSS, and JavaScript for efficient task management. Features include To Do, In Progress, and Done columns, drag-and-drop functionality, task creation and deletion, local storage for data persistence, task counters, scrollable sections, and a clean, modern, mobile-friendly UI.
# 🗂️ Kanban Board

A responsive and interactive **Kanban Board** built using **HTML, CSS, and JavaScript**. This project helps users organize tasks efficiently with a simple and intuitive interface.

## ✨ Features

* 📌 Create new tasks instantly
* 📝 Edit and delete tasks
* 🔄 Drag and drop tasks between columns
* 📋 Three task stages:

  * To Do
  * In Progress
  * Done
* 💾 Local Storage support (tasks remain after refreshing the page)
* 📊 Task count for each column
* 📱 Fully responsive design
* 🎨 Clean and modern user interface
* 📜 Scrollable task lists for better organization
* ⚡ Fast and lightweight with no external frameworks

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 🚀 How to Use

1. Clone the repository.
2. Open `index.html` in your browser.
3. Add new tasks.
4. Move tasks between columns.
5. Delete completed or unwanted tasks.
6. Refresh the page—your tasks are automatically saved using Local Storage.

## 🎯 Purpose

This project was created to strengthen front-end development skills by implementing DOM manipulation, event handling, drag-and-drop functionality, responsive design, and browser storage using only vanilla JavaScript.
